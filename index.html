Getting Started with JavaScript

Steps for getting started

  1. Install the latest version of Node.js
  2. Install an Integrated Devlepomental Environment (IDE)
  3. In VS Code, create a file
  4. Write JS code, save it with .js extension
  5. Open up the terminal
  6. Navigate to the file location
  7. Type node hello.js and hit enter
JavaScript Entry Point

The entry point to a JavaScript application is the first line of code from an interpreted file.

It declares a function then runs it. 

                
                    function(){
                        //functions, variables, etc. 
                        return exports;
                    }();

JavaScript variables are declared in the scope of the function, so different modules do not share names.

JavaScript does not does not need a package keyword.

Printing to the console

To print to the console, we use console.log(text).

This prints the variables inside the log.

                
                    console.log("Hello World")

In this example, we are printing "Hello World" to the console when we run the program.

Object Oriented Programming

JavaScript is a prototype-based object oriented langauage.

This means that has no classes like other object oriented langauages.

An example of this might be an inheritance that inherits objects instead of classes. 

                
                    //Inheritance example
                    class Airplane {
                      constructor(name) {
                        this.name = name;
                      }
                      //method to return the string
                      toString() {
                        return (`Name of the airplane: {this.name}`);
                      }
                    }
                    the class company extends Airplane {
                      constructor(name, color) {
                        //super keyword - refer to the immediate Airplane's class (parent's class) instance variable. 
                        super(name);
                        this.color = color;
                      }
                      toString() {
                        return (`{super.toString()},company color: {this.color}`);
                      }
                    }
                    let company1 = new company('Boeing 737', 'White');
                    console.log(company1.toString());
                    //Name of the airplane: Boeing 737, color: White
